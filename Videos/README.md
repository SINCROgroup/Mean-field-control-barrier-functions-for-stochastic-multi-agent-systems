# Videos

This folder contains representative videos of the two simulation scenarios included in the repository.

## Files

- `coverage.mp4`  
  Example animation of the coverage applicatoin with dangerous disks in a periodic domain.

- `shepherding.mp4`  
  Example animation of the shepherding application with dangerous disks in a periodic domain.


## Purpose

These videos are included only for qualitative visualization of the simulations. They are not required to run the code.
